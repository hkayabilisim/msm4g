§ nbody_box_update()

void nbody_box_update ( SimulationBox *  box,
LinkedList *  list,
double  margin 
)

Update the simulation box for a given list of bodies.

It creates a rectangular there dimensional box to include all of the bodies given in the argument. The width of the box is also enlarged margin percent in each dimension to give some slack to the bodies so that they will stay in the box after time integration.

Todo:
If a body escapes from the box, the box should be updated. But there is a question of how to keep the interior grid points intact. As soon as the box is enlarged, new grid points should be created. MSM may start from scratch whenever the box is updated, but it seems quite waste of time.
Parameters
[in,out]boxThe rectangular simulation box.
[in]listThe list of bodies.
[in]marginThe box is enlarged margin percent (0.10 default).

void nbody_linkedlist_add ( LinkedList *  list,
void *  data 
)

Adds a data pointer to a linked list.

This function is used to add a data pointer to the end of a linked list. Please note that, only the pointer to the data is copied into the linked list. The actual data content should be prepared before calling this function. And for the same reason, when the linked list is deallocated, the original data is not touched. Deallocation of the data should be handled separately.

Parameters
[in,out]listThe linked list to be updated.
[in]dataPointer to the data.

§ nbody_linkedlist_destroy()

void nbody_linkedlist_destroy ( LinkedList *  list)

Deaallocate the linked list.

Since the elements of a linked list are dynamically allocated, the linked list should be destroyed if it is not going to be used anymore. This function deallocates all the elements (`LinkedListElement') in the list.

Warning
This function does not deallocate the actual data pointed in the elements. This is the responsibility of the developer to track down the location of the actual data and deallocate it at the end of the program.
Parameters
[in,out]listThe list to be deallocated.

§ nbody_linkedlist_new()

LinkedList* nbody_linkedlist_new ( )

Creates an empty linked list.

This function allocates an empty linked list and returns its pointer to the caller. A linked list created with this function should be deallocated at the end of the program by calling nbody_linkedlist_destroy function.

In order to add data to the newly creaded linked list, one may use nbody_linkedlist_add function.

Returns
The pointer of the new linked list.
