double msm4g_smoothing_C1 ( double  rho,
int  derivative 
)

C1 smoothing function.

\[ \gamma(\rho) = \left\{ \begin{array}{ll} (3- \rho^2)/2, & \rho \le 1 \\ 1/\rho, & \rho \ge 1. \end{array} \right. \]

Example:

To calculate \( \gamma(1) \) and \( \gamma'(1) \):

double y = msm4g_smoothing_C1(1.0, 0);
double yprime = msm4g_smoothing_C1(1.0, 1);
Parameters
[in]rhoInput variable \( \rho \)
[in]derivativeUse the derivative of specified order.
Returns
Returns \( \gamma(\rho) \).

§ msm4g_d3vector_norm()

double msm4g_d3vector_norm ( D3Vector *  x)

The Euclidean norm of a 3-dimensional double vector.

This function simple calculates the Euclidean norm a vector:

\[ ||x||=\sqrt{\sum_{i=1}^3 x^2_i} \]

Example:

D3Vector x;
double norm;
msm4g_d3vector_set(&x,3.0,0.0,4.0);
norm = msm4g_d3vector_norm(&x)
// norm variable is 5.0 now.
Parameters
[in]xA 3-dimensional double vector.
Returns
The norm of the vector.

int msm4g_math_cantor ( int *  x,
int  n 
)

n-degree generalized Cantor pairing function.

This method is the direct implementation of the n-degree generalized Cantor pairing function defined in Theorem 2.1 in Lisi, M. (2007). Some remarks on the Cantor pairing function. Le Matematiche, 62(1), 55–65.

\[ <x_1,\ldots,x_2> = \sum_{h=1}^n\left\{ \frac{1}{h!}\prod_{j=0}^{h-1} \left[ \left( \sum_{i=1}^hx_i\right)+j\right] \right\} \]

Parameters
[in]xAn integer vector in N^n.
[in]nThe dimension.
Returns
The output of n-degree generalized Cantor pairing function.
