)

Calculates Quasi-interpolation coefficients.

Calculates the coefficients (referred as omegaprime in doi:10.1063/1.4943868) required in Quasi-interpolation. They depend on the order of B-Spline and some adjustable parameter mu.

The output of this function is used to build the stencils. Large mu means more accurate interpolation but at the same it increases the cost.

The callee is responsible to deallocate the array when it is no longer needed.
